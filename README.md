# audiosnip-ffmpeg-mirror

Mirror of [FFmpegKit](https://github.com/arthenica/ffmpeg-kit) iOS audio-variant binaries for AudioSnip's EAS cloud builds. Exists because upstream deleted its GitHub release assets in November 2025 after archiving the project.

## Contents

iOS xcframework binaries are attached as release assets. See [Releases](../../releases).

## License

The FFmpegKit framework and its underlying FFmpeg components are distributed under **LGPL-3.0**. Full license text ships inside each `.xcframework`. FFmpeg source code remains publicly available at [github.com/arthenica/ffmpeg-kit](https://github.com/arthenica/ffmpeg-kit).

## Provenance

Each release note documents the upstream source, the intermediate rebuild source (where applicable), and SHA256 of the file. No modifications are made to the binaries — they are bit-identical copies re-hosted for build reproducibility.
